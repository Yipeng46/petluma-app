# PetLuma App

PetLuma Companion Card Generator, phase 1.

## Included

- Next.js App Router
- TypeScript
- Tailwind CSS
- Home page
- Create page with live companion card preview
- Local image preview only

## Not Included Yet

- Supabase
- Login
- Shopify
- QR codes
- PNG download

## Run

```bash
npm install
npm run dev
```

Open http://localhost:3000.
